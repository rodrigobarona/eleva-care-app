# Webhook Events

This document lists all webhook events available in the Eleva Care platform.

## Event Categories

| Category | Description |
|----------|-------------|
| `booking.*` | Appointment lifecycle events |
| `payment.*` | Payment processing events |
| `expert.*` | Expert-related events |
| `patient.*` | Patient-related events |
| `organization.*` | Organization events |

## Booking Events

### booking.created

Fired when a new booking is initiated (before payment).

```json
{
  "id": "evt_abc123",
  "type": "booking.created",
  "created": "2024-01-15T10:30:00Z",
  "data": {
    "booking": {
      "id": "bkg_def456",
      "status": "pending_payment",
      "expertId": "exp_abc123",
      "serviceId": "svc_xyz789",
      "startTime": "2024-01-20T10:00:00Z",
      "endTime": "2024-01-20T11:00:00Z",
      "timezone": "Europe/Lisbon",
      "amount": 10000,
      "currency": "EUR"
    },
    "patient": {
      "name": "Jane Doe",
      "email": "jane@example.com"
    },
    "expert": {
      "id": "exp_abc123",
      "name": "Dr. Maria Santos"
    }
  }
}
```

### booking.confirmed

Fired when a booking is confirmed (payment successful).

```json
{
  "id": "evt_abc123",
  "type": "booking.confirmed",
  "created": "2024-01-15T10:32:00Z",
  "data": {
    "booking": {
      "id": "bkg_def456",
      "status": "confirmed",
      "expertId": "exp_abc123",
      "startTime": "2024-01-20T10:00:00Z",
      "meetingUrl": "https://meet.eleva.care/bkg_def456"
    },
    "patient": {
      "name": "Jane Doe",
      "email": "jane@example.com"
    },
    "expert": {
      "id": "exp_abc123",
      "name": "Dr. Maria Santos"
    },
    "payment": {
      "id": "pay_ghi789",
      "amount": 10000,
      "currency": "EUR"
    }
  }
}
```

### booking.cancelled

Fired when a booking is cancelled.

```json
{
  "id": "evt_abc123",
  "type": "booking.cancelled",
  "created": "2024-01-16T09:00:00Z",
  "data": {
    "booking": {
      "id": "bkg_def456",
      "status": "cancelled",
      "cancelledAt": "2024-01-16T09:00:00Z",
      "cancelledBy": "patient",
      "cancellationReason": "Schedule conflict"
    },
    "refund": {
      "amount": 10000,
      "status": "processed"
    }
  }
}
```

### booking.rescheduled

Fired when a booking is rescheduled.

```json
{
  "id": "evt_abc123",
  "type": "booking.rescheduled",
  "created": "2024-01-16T10:00:00Z",
  "data": {

      "previousStartTime": "2024-01-20T10:00:00Z",
      "newStartTime": "2024-01-21T14:00:00Z",
      "rescheduledBy": "expert"

  }
}
```

### booking.completed

Fired when a booking is marked as completed.

```json

  "type": "booking.completed",
  "created": "2024-01-20T11:05:00Z",
```

## Payment Events

### payment.succeeded

Fired when a payment is successful.

```json
{
  "id": "evt_abc123",
  "type": "payment.succeeded",
  "created": "2024-01-15T10:31:00Z",
  "data": {
    "payment": {
      "id": "pay_ghi789",
      "amount": 10000,
      "currency": "EUR",
      "status": "succeeded",
      "paymentMethod": "card",
      "bookingId": "bkg_def456"
    },
    "commission": {
      "platform": 800,
      "clinic": 1500,
      "expertNet": 7700
    }
  }
}
```

### payment.failed

Fired when a payment fails.

```json
{
  "id": "evt_abc123",
  "type": "payment.failed",
  "created": "2024-01-15T10:31:00Z",
  "data": {
    "payment": {
      "id": "pay_ghi789",
      "amount": 10000,
      "currency": "EUR",
      "status": "failed",
      "failureCode": "card_declined",
      "failureMessage": "Your card was declined"
    }
  }
}
```

### payment.refunded

Fired when a payment is refunded.

```json
{
  "id": "evt_abc123",
  "type": "payment.refunded",
  "created": "2024-01-16T09:05:00Z",
  "data": {
    "refund": {
      "id": "ref_jkl012",
      "amount": 10000,
      "currency": "EUR",
      "status": "succeeded",
      "paymentId": "pay_ghi789",
      "reason": "booking_cancelled"
    }
  }
}
```

## Expert Events

```json

  "created": "2024-01-15T14:00:00Z",
```

### expert.availability_updated

Fired when an expert updates their availability.

```json

  "data": {

      "removed": ["2024-01-21T10:00:00Z"]
    }
  }
}
```

```json
      }
    }
  }
}
```

## Subscribing to Events

### Via Dashboard

1. Go to **Settings** â†’ **Webhooks**
2. Edit your webhook
3. Select events to receive

### Via API

```bash
curl -X PATCH "https://api.eleva.care/v1/webhooks/wh_abc123" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "events": [
      "booking.confirmed",
      "booking.cancelled",
      "payment.succeeded",
      "payment.refunded"
    ]
  }'
```

## Event Filtering

Filter events by properties:

```bash
curl -X PATCH "https://api.eleva.care/v1/webhooks/wh_abc123" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "events": ["booking.*"],
    "filters": {
      "expertId": ["exp_abc123", "exp_def456"]
    }
  }'
```
