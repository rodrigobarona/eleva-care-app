# API Endpoints

This document provides a complete reference for all Eleva Care API endpoints.

## Base URL

```
Production: https://api.eleva.care/v1
Sandbox:    https://api-sandbox.eleva.care/v1
```

## Experts

### List Experts

Retrieve a paginated list of experts.

```http
GET /experts
```

**Query Parameters:**

| Parameter | Type | Description |
|-----------|------|-------------|
| `category` | string | Filter by category slug |
| `available` | boolean | Only show available experts |
| `language` | string | Filter by language (e.g., "en", "pt") |
| `page` | integer | Page number (default: 1) |
| `limit` | integer | Results per page (max: 50) |

**Example Request:**

```bash
curl -X GET "https://api.eleva.care/v1/experts?category=womens-health&available=true&limit=10" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

**Example Response:**

```json
{
  "success": true,
  "data": [
    {
      "id": "exp_abc123",
      "displayName": "Dr. Maria Santos",
      "title": "OB/GYN Specialist",
      "bio": "Board-certified OB/GYN with 15 years of experience...",
      "profileImage": "https://cdn.eleva.care/profiles/abc123.jpg",
      "categories": ["womens-health", "obstetrics"],
      "languages": ["en", "pt", "es"],
      "averageRating": 4.9,
      "reviewCount": 127,
      "available": true
    }
  ],
  "meta": {
    "pagination": {
      "page": 1,
      "limit": 10,
      "total": 45,
      "pages": 5
    }
  }
}
```

### Get Expert

Retrieve a single expert by ID.

```http
GET /experts/:id
```

**Example Request:**

```bash
curl -X GET "https://api.eleva.care/v1/experts/exp_abc123" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### Get Expert Availability

Retrieve available time slots for an expert.

```http
GET /experts/:id/availability
```

**Query Parameters:**

| Parameter | Type | Description |
|-----------|------|-------------|
| `startDate` | string | Start date (ISO 8601) |
| `endDate` | string | End date (ISO 8601) |
| `serviceId` | string | Filter by specific service |
| `timezone` | string | Response timezone (e.g., "Europe/Lisbon") |

**Example Response:**

```json
{
  "success": true,
  "data": {
    "expertId": "exp_abc123",
    "timezone": "Europe/Lisbon",
    "slots": [
      {
        "date": "2024-01-15",
        "times": [
          { "start": "09:00", "end": "10:00", "available": true },
          { "start": "10:00", "end": "11:00", "available": false },
          { "start": "11:00", "end": "12:00", "available": true }
        ]
      }
    ]
  }
}
```

## Bookings

### Create Booking

Create a new booking.

```http
POST /bookings
```

**Request Body:**

```json
{
  "expertId": "exp_abc123",
  "serviceId": "svc_xyz789",
  "startTime": "2024-01-15T10:00:00Z",
  "patient": {
    "name": "Jane Doe",
    "email": "jane@example.com",
    "phone": "+351912345678",
    "notes": "First-time consultation"
  },
  "timezone": "Europe/Lisbon",
  "paymentMethodId": "pm_card_visa"
}
```

**Example Response:**

```json
{
  "success": true,
  "data": {
    "id": "bkg_def456",
    "status": "pending_payment",
    "expert": {
      "id": "exp_abc123",
      "name": "Dr. Maria Santos"
    },
    "service": {
      "id": "svc_xyz789",
      "name": "Initial Consultation"
    },
    "startTime": "2024-01-15T10:00:00Z",
    "endTime": "2024-01-15T11:00:00Z",
    "timezone": "Europe/Lisbon",
    "amount": 10000,
    "currency": "EUR",
    "paymentIntent": {
      "clientSecret": "pi_xxx_secret_yyy"
    },
    "createdAt": "2024-01-10T15:30:00Z"
  }
}
```

### Get Booking

Retrieve booking details.

```http
GET /bookings/:id
```

### List Bookings

List bookings for your organization.

```http
GET /bookings
```

**Query Parameters:**

| Parameter | Type | Description |
|-----------|------|-------------|
| `status` | string | Filter by status (pending, confirmed, completed, cancelled) |
| `expertId` | string | Filter by expert |
| `startDate` | string | Filter by start date |
| `endDate` | string | Filter by end date |
| `page` | integer | Page number |
| `limit` | integer | Results per page |

### Cancel Booking

Cancel an existing booking.

```http
POST /bookings/:id/cancel
```

**Request Body:**

```json
{
  "reason": "Patient requested cancellation",
  "refundAmount": 10000
}
```

### Reschedule Booking

Reschedule a booking to a new time.

```http
POST /bookings/:id/reschedule
```

**Request Body:**

```json
{
  "newStartTime": "2024-01-16T14:00:00Z",
  "notifyPatient": true
}
```

## Webhooks

### List Webhook Subscriptions

```http
GET /webhooks
```

### Create Webhook Subscription

```http
POST /webhooks
```

**Request Body:**

```json
{
  "url": "https://yourapp.com/webhooks/eleva",
  "events": [
    "booking.created",
    "booking.confirmed",
    "booking.cancelled",
    "payment.succeeded"
  ],
  "secret": "whsec_xxxxxxxx"
}
```

### Delete Webhook Subscription

```http
DELETE /webhooks/:id
```

## Pagination

All list endpoints support pagination:

```json
{
  "meta": {
    "pagination": {
      "page": 1,
      "limit": 20,
      "total": 157,
      "pages": 8,
      "hasNext": true,
      "hasPrev": false
    }
  }
}
```

Navigate pages using query parameters:

```http
GET /experts?page=2&limit=20
```

## Filtering

Common filter parameters:

| Parameter | Type | Description |
|-----------|------|-------------|
| `search` | string | Full-text search |
| `sort` | string | Sort field (e.g., "createdAt") |
| `order` | string | Sort order ("asc" or "desc") |
| `fields` | string[] | Specific fields to return |
