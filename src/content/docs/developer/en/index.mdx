```

│                 │◄──►│   API           │◄──►│   Services      │
│ • Web App       │    │ • REST API      │    │ • Stripe        │
```

### Authentication

All API requests require authentication via API key:

```bash
curl -X GET "https://api.eleva.care/v1/experts" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

### Rate Limits

| Plan | Requests/Minute | Requests/Day |
|------|-----------------|--------------|
| Enterprise | 100 | 10,000 |
| Custom | Negotiable | Negotiable |

## Common Use Cases

<Tabs items={['Embed Booking', 'CRM Sync', 'Custom Calendar']}>
  <Tab value="Embed Booking">
    **Embed Booking Widget**

```html
<script src="https://cdn.eleva.care/embed.js"></script>
<div
  id="eleva-booking"
  data-org-id="YOUR_ORG_ID"
  data-expert-id="EXPERT_ID"
  data-theme="light"
></div>
```

  </Tab>
  <Tab value="CRM Sync">
    **Sync Bookings to Your CRM**

```typescript
// Webhook handler for booking events
app.post('/webhooks/eleva', (req, res) => {
  const event = req.body;
  
  if (event.type === 'booking.confirmed') {
    const { booking, patient, expert } = event.data;
    
    // Sync to your CRM
    await crm.createAppointment({
      patientEmail: patient.email,
      expertName: expert.name,
      scheduledAt: booking.startTime,
      service: booking.serviceName,
    });
  }
  
  res.status(200).json({ received: true });
});
```

  </Tab>
  <Tab value="Custom Calendar">
    **Build Custom Availability Calendar**

```typescript
// Fetch expert availability
const availability = await fetch(
  'https://api.eleva.care/v1/experts/EXPERT_ID/availability',
  {
    headers: {
          Authorization: `Bearer ${API_KEY}`,
        },
    },
);

const slots = await availability.json();
// Render custom calendar with available slots
```

  </Tab>
</Tabs>

### Quick Example

```typescript
import { ElevaCare } from '@eleva-care/sdk';

const client = new ElevaCare({
  apiKey: process.env.ELEVA_API_KEY,
});

// List experts
const experts = await client.experts.list({
  category: 'womens-health',
  available: true,
});

// Create a booking
const booking = await client.bookings.create({
  expertId: 'exp_123',
  serviceId: 'svc_456',
  startTime: '2024-01-15T10:00:00Z',
  patient: {
    name: 'Jane Doe',
    email: 'jane@example.com',
  },
});
```
