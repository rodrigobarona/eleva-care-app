# Analytics Dashboard

Your Partner Dashboard provides comprehensive analytics to help you track performance, optimize operations, and grow your workspace.

## Dashboard Overview

Access your analytics at **Partner Dashboard** → **Analytics**.

### Key Metrics at a Glance

| Metric | Description |
|--------|-------------|
| **Total Revenue** | Gross booking amount |
| **Your Earnings** | Marketing fees collected |
| **Active Experts** | Experts with bookings this period |
| **Bookings** | Total appointments scheduled |
| **Utilization** | Percentage of available slots booked |
| **Average Rating** | Patient satisfaction score |

## Booking Analytics

### Booking Volume

Track appointment activity:

| Metric | Description |
|--------|-------------|
| **Total Bookings** | All scheduled appointments |
| **Completed** | Successfully completed sessions |
| **Cancelled** | Cancelled appointments (with reasons) |
| **No-Shows** | Patients who didn't attend |

### Booking Funnel

Understand your conversion:

```
Profile Views → Booking Started → Booking Completed → Attended
    1,000    →     200 (20%)    →     150 (75%)     → 140 (93%)
```

### Booking Sources

See where patients come from:

- Direct (your workspace profile)
- Search (Eleva Care search)
- Expert profile
- External referral

<Callout type="info">
  **Pro Tip**: High search traffic indicates strong SEO. Low direct traffic might mean you need
  more external marketing.
</Callout>

## Using Analytics Effectively

<Tabs items={['Weekly', 'Monthly', 'Quarterly']}>
  <Tab value="Weekly">
    **Weekly Review** - Check every week:

    1. **Revenue**: Are you on track for monthly goals?
    2. **Bookings**: Any unusual cancellation patterns?
    3. **Ratings**: Any negative feedback to address?
    4. **Availability**: Any scheduling gaps?

  </Tab>
  <Tab value="Monthly">
    **Monthly Deep Dive** - Analyze once a month:

    1. **Growth Trends**: Are you growing? Where?
    2. **Expert Performance**: Who needs support? Recognition?
    3. **Service Mix**: Which services should you expand?
    4. **Patient Feedback**: What themes emerge?

  </Tab>
  <Tab value="Quarterly">
    **Quarterly Planning** - Plan based on:

    1. **Revenue Goals**: Set realistic targets
    2. **Team Expansion**: When to add experts?
    3. **Marketing Strategy**: What's working?
    4. **Service Innovation**: New services to offer?

  </Tab>
</Tabs>
